# display属性_visibility

## 行内元素转换为块级元素

* `display`
  * 可以指定元素所生产的框的类型
    * `inline`: 行内元素
    * `block`: 块元素
    * `inline-block`: 行内块元素 (既有行内元素特点-不独占一行， 又有块元素特点-可以设置宽高)
      * 但是内联块，直接会有空白折叠
    * `none`: 元素不显示

## visibility

1. 可以设置元素的显示的状态
    
2. 可选值
    
    1. `visible`: 默认值，元素正常显示
    2. `hidden`: 不显示元素，但是元素依然占据位置
