# 数据库
* 数据库是在服务器上面的，而我们的计算机现在可以作为服务器，所以我们有了数据库。
* 数据库基础
    * 建立数据库，建立表
    * 数据库里面有表，表里面有字段，字段就是表头，每一条数据就是一行一行的数据

## PHP连接数据库和SQL语句
* SQL 全称是Structor Query Language结构化查询语言，用于操作数据库
* SQL要读取数据，语法就是
    * `SELECT * FROM 表名字;`
